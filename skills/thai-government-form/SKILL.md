# หนังสือราชการไทย (Thai Government Letters)

## Overview
หนังสือราชการต้องเป็นไปตาม **ระเบียบสำนักนายกรัฐมนตรีว่าด้วยงานสารบรรณ พ.ศ. 2526** (และฉบับแก้ไขเพิ่มเติม) — โครงสร้าง, คำขึ้นต้น, คำลงท้าย, และตำแหน่งของส่วนประกอบทุกอย่างถูกกำหนดไว้ตายตัว. Thai official letters follow ระเบียบสารบรรณ 2526; every block, salutation, and closing phrase has a prescribed form and incorrect rank-of-address can cause the letter to be returned without action.

## When to use
- ผู้ใช้ต้องเขียนหนังสือถึงหน่วยงานราชการ/รัฐวิสาหกิจ
- ผู้ใช้ต้องร่างคำร้อง, หนังสือมอบอำนาจ, ใบลา, หนังสือรับรอง, หรือหนังสือเชิญ
- ผู้ใช้ต้องเลือกคำขึ้นต้น (เรียน/กราบเรียน/กราบบังคมทูล) ให้ถูกต้องตามยศ/ตำแหน่งของผู้รับ

## Conventions / Rules

### โครงสร้างหนังสือภายนอก (External letter format — per ระเบียบสารบรรณ 2526)

```
[ตราครุฑ / หัวกระดาษหน่วยงาน — กลางหน้า]

ที่ {{หน่วยงาน}} {{เลขลำดับ}}                              {{ที่อยู่ผู้ส่ง}}

                                                          {{วันที่ เดือน พ.ศ.}}

เรื่อง   {{หัวเรื่องสั้น เป็นวลีคำนาม ไม่มีกริยา}}

เรียน   {{ตำแหน่งผู้รับ}}

อ้างถึง  {{หนังสือที่อ้างอิง — ถ้ามี}}

สิ่งที่ส่งมาด้วย   {{รายการเอกสารแนบ — ถ้ามี}}

         {{ย่อหน้าที่ 1 — ขึ้นต้นด้วย "ด้วย..." หรือ "ตามที่..."}}

         {{ย่อหน้าที่ 2 — รายละเอียด/เนื้อหา}}

         {{ย่อหน้าปิด — "จึงเรียนมาเพื่อโปรดทราบ" / "จึงเรียนมาเพื่อโปรดพิจารณา" / "จึงเรียนมาเพื่อโปรดดำเนินการต่อไป"}}

                                            ขอแสดงความนับถือ

                                            {{ลายเซ็น}}

                                            ({{ชื่อ-นามสกุล พร้อมคำนำหน้า}})
                                            {{ตำแหน่ง}}
                                            {{หน่วยงาน}}

{{หน่วยที่รับผิดชอบ / เบอร์โทรศัพท์ — มุมล่างซ้าย}}
```

### เลขที่หนังสือ (Letter reference number)
- รูปแบบ: **ที่ {{รหัสกระทรวง}} {{รหัสกรม}}/{{เลขลำดับ}}**
- ตัวอย่าง: **ที่ ศธ ๐๔๐๐๒/๑๒๓๔** (ศธ = กระทรวงศึกษาธิการ; ๐๔๐๐๒ = รหัสกรม; ๑๒๓๔ = เลขลำดับของหน่วยงาน)
- หน่วยงานเอกชนใช้ **ที่ {{ตัวอักษรย่อบริษัท}} {{ปี}}/{{เลขลำดับ}}** เช่น "ที่ บ.ก. ๒๕๖๙/๐๐๑"

### ลำดับชั้นของคำขึ้นต้น (Salutation hierarchy)

| ผู้รับ | คำขึ้นต้น | คำลงท้าย |
|---|---|---|
| **พระมหากษัตริย์ / สมเด็จพระบรมราชินี** | กราบบังคมทูล (ผ่านราชเลขานุการ) | ควรมิควรแล้วแต่จะทรงพระกรุณาโปรดเกล้าโปรดกระหม่อม / ด้วยเกล้าด้วยกระหม่อม ขอเดชะ |
| **พระบรมวงศานุวงศ์ชั้นสมเด็จเจ้าฟ้า** | ขอพระราชทานกราบบังคมทูล / กราบทูล | ควรมิควรแล้วแต่จะโปรดเกล้าฯ |
| **สมเด็จพระสังฆราช** | กราบทูล | ควรมิควรแล้วแต่จะโปรด |
| **พระสงฆ์ชั้นพระราชาคณะขึ้นไป / พระสงฆ์ทั่วไป** | นมัสการ | ขอนมัสการด้วยความเคารพอย่างยิ่ง |
| **นายกรัฐมนตรี, ประธานองคมนตรี, ประธานรัฐสภา, ประธานวุฒิสภา, ประธานสภาผู้แทนราษฎร, ประธานศาลฎีกา, ประธานศาลรัฐธรรมนูญ, ประธานศาลปกครองสูงสุด, รัฐบุรุษ** (และคู่สมรสของบุคคลเหล่านี้) | กราบเรียน | ขอแสดงความนับถืออย่างยิ่ง |
| **รัฐมนตรี, ปลัดกระทรวง, อธิบดี, ผู้ว่าราชการจังหวัด, นายอำเภอ, ผอ., หัวหน้า, นาย/นาง/นางสาว ทั่วไป** | เรียน | ขอแสดงความนับถือ |

> **สำคัญ:** การเลือกผิดระดับสามารถทำให้หนังสือถูกส่งกลับโดยไม่ดำเนินการ

### ประเภทเอกสารราชการที่พบบ่อย (Common form types)

| ประเภท | ความหมาย | คำขึ้นต้นในเนื้อความ |
|---|---|---|
| **คำร้อง** | คำขอจากประชาชน/บุคคลภายนอก | "ข้าพเจ้า {{ชื่อ}} {{เลขบัตร}} ขอยื่นคำร้องต่อ..." |
| **หนังสือมอบอำนาจ** | มอบหมายให้บุคคลอื่นดำเนินการแทน | "ข้าพเจ้า ... ขอมอบอำนาจให้ ... ดำเนินการ..." |
| **ใบลา (ป่วย/กิจ/คลอด/บวช/พักผ่อน)** | ขอลาหยุดงาน | "ข้าพเจ้า ... ขอลา{{ประเภท}} ตั้งแต่วันที่ ... ถึงวันที่ ..." |
| **หนังสือรับรอง** | ยืนยันข้อเท็จจริง/สถานะ | "หนังสือฉบับนี้ให้ไว้เพื่อรับรองว่า ..." |
| **หนังสือเชิญ** | เชิญเข้าร่วมงาน/ประชุม | "ด้วย {{หน่วยงาน}} จะจัด ... จึงขอเรียนเชิญ ..." |

### กฎโทน (Tone rules)
- **สรรพนามบุรุษที่ 1:** ใช้ **ข้าพเจ้า** (กลาง), **กระผม** (ชายทางการ), **ดิฉัน** (หญิงทางการ) — **ห้าม** ผม/หนู/เรา
- **ห้ามใช้ "ครับ/ค่ะ"** ในหนังสือราชการเด็ดขาด
- **ห้ามอิโมจิ, ห้ามเครื่องหมายอัศเจรีย์ (!)** 
- **คำศัพท์:** ใช้ **ดำเนินการ** (ไม่ใช่ "operate"), **พิจารณา** (ไม่ใช่ "consider"), **อนุเคราะห์** (ขอความช่วยเหลือ), **โปรด** (สุภาพ)
- **กริยา passive:** "ขอความอนุเคราะห์" / "ใคร่ขอ" / "หวังเป็นอย่างยิ่งว่าจะได้รับ..." — ไม่ใช้ active voice ตรงๆ
- **ปิดย่อหน้าสุดท้าย:** ต้องเป็น "จึงเรียนมาเพื่อโปรด..." เสมอ
- **วันที่:** รูปแบบเต็ม "{{วันที่}} {{เดือนเต็ม}} {{พ.ศ.}}" เช่น "วันที่ ๒ พฤษภาคม ๒๕๖๙"
- **ตัวเลข:** สามารถเป็นเลขไทย (๒๕๖๙) หรือเลขอารบิก (2569) — **ต้องสม่ำเสมอตลอดเอกสาร**

## Templates / Examples

### Example 1: หนังสือขอความอนุเคราะห์ถึงอธิบดี
```
ที่ บ.ก. ๒๕๖๙/๐๑๒

                                          บริษัท สมหวัง จำกัด
                                          ๑๒๓ ถนนพระราม ๔ แขวงคลองเตย
                                          เขตคลองเตย กรุงเทพมหานคร ๑๐๑๑๐

                                          วันที่ ๒ พฤษภาคม ๒๕๖๙

เรื่อง   ขอความอนุเคราะห์ข้อมูลสถิติการนำเข้าสินค้าประจำปี ๒๕๖๘

เรียน   อธิบดีกรมศุลกากร

         ด้วยบริษัท สมหวัง จำกัด มีความประสงค์จะจัดทำรายงานการวิเคราะห์ตลาดสินค้านำเข้า
ประจำปี ๒๕๖๘ เพื่อใช้ประกอบการวางแผนกลยุทธ์ทางธุรกิจ

         ในการนี้ บริษัทฯ ใคร่ขอความอนุเคราะห์ข้อมูลสถิติการนำเข้าสินค้าในหมวด {{หมวด}}
ตั้งแต่เดือนมกราคม ถึง เดือนธันวาคม ๒๕๖๘ เพื่อใช้ในการศึกษาวิจัยดังกล่าว

         จึงเรียนมาเพื่อโปรดพิจารณาให้ความอนุเคราะห์ บริษัทฯ ขอขอบคุณเป็นอย่างสูง
มา ณ โอกาสนี้

                                          ขอแสดงความนับถือ


                                          (นายสมชาย ใจดี)
                                          กรรมการผู้จัดการ
                                          บริษัท สมหวัง จำกัด

ฝ่ายการตลาด
โทร. ๐๒-๑๒๓-๔๕๖๗
```


## Common mistakes
- ใช้ "ครับ/ค่ะ" ในหนังสือราชการ (ห้ามเด็ดขาด)
- เลือกคำขึ้นต้นผิดระดับ (เช่น ใช้ "เรียน" กับนายกรัฐมนตรี → ต้อง "กราบเรียน") — หนังสืออาจถูกส่งกลับ
- ใช้ "กราบเรียน" กับพระสงฆ์ — ต้อง "นมัสการ" (พระทั่วไป) หรือ "กราบทูล" (สมเด็จพระสังฆราช)
- ใช้คำศัพท์ภาษาอังกฤษทับศัพท์ ("operate", "implement", "process") — ต้องใช้ "ดำเนินการ"
- ผสมเลขไทย/อารบิกในเอกสารเดียวกัน
- ลืมบรรทัด **เรื่อง:** — เป็นองค์ประกอบบังคับ
- ลืมปิดด้วย "จึงเรียนมาเพื่อโปรด..." — สูตรปิดที่ขาดไม่ได้
- ใช้สรรพนาม "ผม/หนู/เรา" แทน "ข้าพเจ้า/กระผม/ดิฉัน"
- หัวเรื่องเป็นประโยค (มีกริยา) — ต้องเป็น**วลีคำนามสั้น** เช่น "ขอความอนุเคราะห์ข้อมูล..." ไม่ใช่ "บริษัทฯ ต้องการข้อมูล"
- ทิ้งช่อง "ที่ ..." ว่าง — ต้องมีเลขที่หนังสือเสมอ (รูปแบบ ที่ {{รหัส}}/{{ลำดับ}})
